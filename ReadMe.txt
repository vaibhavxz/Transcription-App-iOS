The app is built in Swift and UIKit.
Apple's AVKit is used for the video player the JSON file provided is parsed for generating transcriptions.
The Video Player and Transcription Classes are segregated.
The transcriptions are shown in a label by updating sentences and the current word gets highlighted in yellow.

Instructions to run the app:
Hit build in Xcode. Click on Play Video. The Video and JSON are bundled in the app so a change in path is not required.
